# A. Background（背景・狙い）

> ブレ防止の憲法。ここが揺れたら全部崩れる。

---

## 1. 背景・課題（Why）

家賃、保険、通信費、サブスクリプションなどの固定費は、生活しているだけで継続的に発生する支出である。

しかしこれらの支出は、日々の行動と直接結びつけて認識されにくく、「何もしない時間」にも生活コストが発生しているという感覚を持ちにくい。

その結果として、

- SNSの閲覧
- 動画視聴
- ゲーム
- 目的のないネット閲覧

などの時間消費が「無料の時間」として扱われてしまう傾向がある。

しかし実際には、**生活コストは常に時間と共に発生している**。

時間は有限であり、生活している限りコストが乗っているにもかかわらず、その事実が数値として可視化されていないことが問題である。

本システムは、固定費を時間単位に換算して可視化することで、

- **時間の価値を数値として認識できる状態**
- **自分の時間の重みを実感できる状態**

を作ることを目的とする。

---

## 2. ゴール（成果物）

本ツールを使用した結果、以下の状態が達成されていることを成功とする。

| 領域 | 達成状態 |
|------|----------|
| **固定費管理** | 固定費をカテゴリごとに登録できる／編集・削除できる |
| **計算表示** | 月額合計・1日・1時間・1分あたりのコストが自動計算される |
| **データ保持** | 入力データがブラウザに保存され、再訪時に復元される |
| **アプリ形式** | URLでアクセス可能／PWAとしてインストール可能／独立ウィンドウで起動可能 |

**前提・制約**

- 本ツールは数値表示のみを行う
- AIによる判断・評価は行わない

---

## 3. 利用者・利用シーン

### 利用者（Who）

| 区分 | 対象 |
|------|------|
| 主な利用者 | 個人（自分自身、友人・知人） |
| 想定リテラシー | ITに詳しくない人、アプリ操作に慣れていない人でも利用できる設計 |

### 利用シーン（When / Where）

| 項目 | 内容 |
|------|------|
| **利用タイミング** | 固定費を見直したいとき／作業・勉強を始める前／無駄な時間を使ったと感じたとき |
| **利用環境** | PCブラウザ（Mac / Windows）、スマートフォン |
| **利用頻度** | 毎日使うことは前提としない。必要なときに数分使うツール |

---

## 4. スコープ（やる/やらない）

### やること（MVP）

- 固定費の登録
- 固定費の編集
- 固定費の削除
- 月額固定費合計の算出
- 時間コスト表示（1日・1時間・1分）
- データ保存（localStorage）
- データ復元
- Web公開
- PWA対応
- **スマホでの操作**（レスポンシブ対応）

### やらないこと（スコープ外）

| カテゴリ | 対象 |
|----------|------|
| 家計・費目 | 変動費管理、家計簿機能 |
| 履歴・ログ | 履歴管理、行動ログ |
| 可視化 | グラフ表示 |
| 外部連携 | 銀行連携、クレジットカード連携 |
| 機能拡張 | AI機能、通知機能 |
| アカウント | ユーザーアカウント、データ同期 |
