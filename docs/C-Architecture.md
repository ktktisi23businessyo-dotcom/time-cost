## 1. 画面構成 / 画面遷移

```
┌─────────────────────────────────────────┐
│  1日のお金 - 固定費を時間コストに換算    │
├─────────────────────────────────────────┤
│  【時間コスト表示エリア】                │
│  月額: ○○円  1日: ○○円  1時間: ○○円  1分: ○○円 │
├─────────────────────────────────────────┤
│  【固定費登録フォーム】                  │
│  カテゴリ / 項目名 / 月額 / メモ         │
│  [登録]                                 │
├─────────────────────────────────────────┤
│  【固定費一覧】                          │
│  カテゴリ | 項目名 | 月額 | [編集][削除] │
│  ...                                    │
└─────────────────────────────────────────┘
```

**画面遷移**

- 単一ページ構成（SPAではない、1画面のみ）
- モーダル等は削除確認時のみ使用想定

**スマホ対応**

- viewport meta タグでモバイル表示最適化
- レスポンシブCSS（タッチ操作しやすいボタンサイズ、スクロール可能な一覧）

## 2. データモデル（エンティティと主要項目）

### 固定費（FixedCost）

| 項目 | 型 | 必須 | 備考 |
|------|-----|------|------|
| id | string | ○ | 一意識別用（UUID または タイムスタンプ） |
| category | string | ○ | 家賃/保険/通信費/サブスク/車関連/その他 |
| name | string | △ | 「その他」のとき必須、それ以外は任意 |
| amount | number | ○ | 月額金額（整数円） |
| memo | string | - | 任意 |

### localStorage 保存形式

```json
[
  {"id":"xxx","category":"家賃","name":"","amount":70000,"memo":""},
  {"id":"yyy","category":"通信費","name":"","amount":5000,"memo":""}
]
```

**保存キー例**: `fixed-costs` または `ichinichi-okane-data`

## 3. 外部連携

| 対象 | 役割 | 流れ |
|------|------|------|
| **外部API** | なし | - |
| **LINE/Slack/Sheets** | なし | - |

## 5. ログ方針

| 対象 | 出力内容 | 出力先 |
|------|----------|--------|
| **エラー** | バリデーションエラー、想定外の例外 | console.error（開発時） |
| **デバッグ** | 保存/読み込みの成否（必要時のみ） | console.log（開発時、本番では削除可） |
| **ユーザー** | 不正入力時のメッセージ | 画面上に表示 |

**本ツールはサーバーサイドを持たないため、サーバーログはなし。**
